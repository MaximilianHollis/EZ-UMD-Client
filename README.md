# 📅 EZ-UMD 🏗️

## Easy peasy schedule builder 🏗️ for UMD* students.

## 💻 Run Locally 

Clone the project  💾

```bash
git clone https://github.com/MaximilianHollis/EZ-UMD-Client.git
```

Go to the project directory 

```bash
cd EZ-UMD-Client
```

### Install dependencies 🚚

With [cotton](https://cotton.js.org/) (much faster) 😎

```bash
cotton install
```

With yarn 

```bash
yarn
```
